# 数学記号解説コマンド

引数として渡された数学記号 `{{args}}` について、以下の4点を**必ずセット**で日本語で解説してください。

1. **読み方**: 英語と日本語の両方
2. **意味**: 直感的説明 + 厳密な定義
3. **Python での書き方**: NumPy / SymPy / 標準ライブラリ
4. **小さな具体例**: 数字を入れた最小例 + 期待出力
5. **応用先**: ML/AI/統計のどの場面で頻出するか

最後に `glossary/symbol_reference.md` に追記すべきか確認してください。
