# 論理記号

論文の定義・定理で頻出。慣れると数式が「文章」のように読めるようになる。

## 量化子（最重要）

| 記号 | 読み方 | 意味 | Python |
|---|---|---|---|
| `∀` | "for all" / 「すべての」「任意の」 | すべての要素について | `all(...)` |
| `∃` | "there exists" / 「存在する」 | 少なくとも1つ存在 | `any(...)` |
| `∃!` | "there exists unique" / 「ただ1つ存在」 | ちょうど1つ存在 | （カウントして判定） |
| `s.t.` または `:` | "such that" / 「〜であるような」 | 条件付与 | `if`（内包表記） |

### 例

数学:
```
∀x ∈ ℝ,  x² ≥ 0
```
読み方: 「すべての実数 x について、x² は 0 以上」

```
∃x ∈ ℝ s.t. x² = 2
```
読み方: 「ある実数 x が存在して、x² = 2 を満たす」

Python:
```python
import numpy as np

xs = np.linspace(-10, 10, 1000)

# ∀x, x² ≥ 0
print(all(x**2 >= 0 for x in xs))    # True

# ∃x s.t. x² = 2（近似）
print(any(np.isclose(x**2, 2.0, atol=0.01) for x in xs))  # True
```

## 論理演算子

| 記号 | 読み方 | 意味 | Python |
|---|---|---|---|
| `∧` | "and" / 「かつ」 | 論理積 | `and`（スカラー）、`&`（配列） |
| `∨` | "or" / 「または」 | 論理和 | `or`、`\|` |
| `¬` または `~` | "not" / 「ではない」 | 否定 | `not`、`~` |
| `⇒` または `→` | "implies" / 「ならば」 | 含意 | `not P or Q` で表現 |
| `⇔` または `↔` | "if and only if" / 「同値」 | 同値 | `P == Q`（bool） |

### 含意 `⇒` の注意

「P ⇒ Q」は **P が偽なら全体は真** という直感に反するルール。

| P | Q | P ⇒ Q |
|---|---|---|
| 真 | 真 | 真 |
| 真 | 偽 | **偽** |
| 偽 | 真 | 真 |
| 偽 | 偽 | 真 |

Python:
```python
def implies(P: bool, Q: bool) -> bool:
    """P ⇒ Q の真理値."""
    return (not P) or Q

print(implies(True, True))    # True
print(implies(True, False))   # False
print(implies(False, True))   # True
print(implies(False, False))  # True
```

## 同値 `⇔` の意味

「P ⇔ Q」は「P と Q は同時に真、または同時に偽」。
数学の定理で「〜であることと、…であることは同値である」と言うときに使う。

```python
def iff(P: bool, Q: bool) -> bool:
    """P ⇔ Q の真理値."""
    return P == Q
```

## 実例（読み解く練習）

数学:
```
∀ε > 0, ∃δ > 0 s.t. |x − a| < δ ⇒ |f(x) − f(a)| < ε
```

読み方:
「任意の正の数 ε について、ある正の数 δ が存在して、
 |x − a| < δ ならば |f(x) − f(a)| < ε」

これは関数 `f` が点 `a` で**連続である**ことの定義です（ε-δ論法）。

## ハマりポイント

- `∀` と `∃` の順序を変えると意味が変わる:
  - `∀x ∃y, y > x` → 任意の x に対し、それより大きい y が存在する（真）
  - `∃y ∀x, y > x` → ある特定の y が、すべての x より大きい（偽）
- 自然言語の「ならば」と論理学の `⇒` は微妙にニュアンスが違う
- 否定は丁寧に: `¬(∀x, P(x))` ⇔ `∃x, ¬P(x)`
