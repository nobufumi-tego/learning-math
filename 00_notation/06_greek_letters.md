# ギリシャ文字とよくある慣習

数学・物理・統計・MLで頻出。**「この文字はだいたいこういう意味」**という慣習を覚えると論文が読みやすくなる。

## ギリシャ文字一覧

| 小文字 | 大文字 | 読み方 | よくある意味（数学/ML/統計） |
|---|---|---|---|
| α | Α | alpha / アルファ | 学習率、有意水準、係数 |
| β | Β | beta / ベータ | 回帰係数、第二パラメータ |
| γ | Γ | gamma / ガンマ | 割引率（強化学習）、ガンマ関数 |
| δ | Δ | delta / デルタ | 微小量、誤差、ディラックのδ関数 |
| ε | Ε | epsilon / イプシロン | 微小な正数、許容誤差、ノイズ項 |
| ζ | Ζ | zeta / ゼータ | ゼータ関数 |
| η | Η | eta / イータ | 学習率（αの代わり） |
| θ | Θ | theta / シータ | 角度、モデルパラメータ |
| ι | Ι | iota / イオタ | （ほぼ使わない） |
| κ | Κ | kappa / カッパ | 条件数、カーネルパラメータ |
| λ | Λ | lambda / ラムダ | 固有値、正則化係数、ポアソン分布の率 |
| μ | Μ | mu / ミュー | 平均、期待値 |
| ν | Ν | nu / ニュー | 自由度、振動数 |
| ξ | Ξ | xi / クシー（グザイ） | 補助変数、確率変数 |
| ο | Ο | omicron / オミクロン | （ほぼ使わない） |
| π | Π | pi / パイ | 円周率、確率分布、総乗（大文字） |
| ρ | Ρ | rho / ロー | 相関係数、密度 |
| σ | Σ | sigma / シグマ | 標準偏差、活性化関数、総和（大文字） |
| τ | Τ | tau / タウ | 時定数、温度パラメータ |
| υ | Υ | upsilon / ユプシロン | （ほぼ使わない） |
| φ | Φ | phi / ファイ | 特性関数、標準正規累積分布関数（大文字 Φ） |
| χ | Χ | chi / カイ | カイ二乗分布 |
| ψ | Ψ | psi / プサイ | 波動関数 |
| ω | Ω | omega / オメガ | 角速度、サンプル空間（大文字 Ω） |

## 機械学習で特によく出るもの（優先暗記）

| 文字 | 典型的な意味 | 例 |
|---|---|---|
| `θ` | モデルパラメータ | `θ = (w, b)` |
| `α`, `η` | 学習率 | `θ ← θ − η ∇L(θ)` |
| `λ` | 正則化係数 | `L + λ\|\|w\|\|²` |
| `μ` | 平均 | 正規分布 N(μ, σ²) |
| `σ` | 標準偏差 | 正規分布 N(μ, σ²) |
| `ε` | 微小量 / ノイズ | `y = f(x) + ε` |
| `Σ` | 共分散行列（大文字） | `N(μ, Σ)` |
| `∇` | 勾配（ナブラ） | `∇L(θ)` |

## Python での書き方

ギリシャ文字は変数名としては英語表記が一般的:

```python
# 数学: y = α x + β
def linear(x: float, alpha: float, beta: float) -> float:
    return alpha * x + beta

# 数学: θ ← θ − η ∇L
theta = theta - eta * grad_L
```

### Unicode 変数名

Python では Unicode 変数名が使えるが、可読性のため**通常は英語表記を推奨**:

```python
# OK だが推奨しない
α = 0.01
θ = np.zeros(10)

# 推奨
alpha = 0.01
theta = np.zeros(10)
```

## ナブラ ∇ と関連記号

| 記号 | 読み方 | 意味 |
|---|---|---|
| `∇f` | "nabla f" / 「ナブラ f」 | f の勾配（多変数の微分） |
| `∇·F` | "divergence" / 「発散」 | ベクトル場の発散 |
| `∇×F` | "curl" / 「回転」 | ベクトル場の回転 |
| `∇²` または `Δ` | "Laplacian" / 「ラプラシアン」 | 二階偏微分の和 |
| `∂` | "partial" / 「ラウンドディー」 | 偏微分 |

```python
import sympy as sp

x, y = sp.symbols("x y")
f = x**2 + 3 * y

# ∂f/∂x
print(sp.diff(f, x))  # 2*x

# ∇f = (∂f/∂x, ∂f/∂y)
grad = [sp.diff(f, v) for v in (x, y)]
print(grad)  # [2*x, 3]
```

## 大文字ギリシャ文字の特殊な使い方

| 記号 | 用途 |
|---|---|
| `Σ` | 総和（演算子） / 共分散行列 |
| `Π` | 総乗（演算子）/ 確率分布の積 |
| `Δ` | 差分 / ラプラシアン |
| `Γ` | ガンマ関数 |
| `Φ` | 標準正規累積分布関数 |
| `Λ` | 対角行列 |
| `Ω` | 全体（サンプル空間） |

## ハマりポイント

- 同じ文字でも分野で意味が違う（例: `σ` は統計だと標準偏差、ML だと活性化関数 sigmoid）
- 大文字 `Σ` は「演算子」と「共分散行列」を**文脈で判別**
- ε は「小さな正の数」の慣例だが、強化学習の「ε-greedy」では「探索確率」
- ν と v は字形が似ていて混同しがち
