# 用語集・記号リファレンス

学習中の「逆引き」用。論文・教科書・生成AI出力で出てきた記号や用語を即座に調べる。

## ファイル

- `symbol_reference.md` — 数学記号と Python 対応の一覧
- `jp_en_terms.md` — 日英用語対訳（英語論文を読むときの辞書）

## 使い方

### 記号がわからないとき
1. `symbol_reference.md` を Ctrl+F で検索
2. 載っていなければ Claude Code で `/project:explain-symbol <記号>`
3. 学んだら自分でこのファイルに追記
