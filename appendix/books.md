# 📕 書籍ガイド — 評価が高くて買って損しない 50 冊

各書籍について **「誰におすすめか」「なぜ良いか」「どこで買えるか」** を明記。
日本語/英語、入門/中級/上級、有料/無料 を併記。

📌 **凡例**:
- 🆓 = 無料公開 (PDF や Web で読める)
- 💰 = 有料書籍
- 🇯🇵 = 日本語
- 🇬🇧 = 英語
- 🌟 = 特に推奨 (この分野の定番)

## 1. 数学アレルギー解消のための入門書

### 数学ガール
- **著者**: 結城 浩
- **出版**: SB Creative
- 💰 🇯🇵 🌟
- **対象**: 高校数学に苦手意識がある人〜大学生
- **おすすめ理由**: ストーリー仕立てで読みやすく、難しい概念が「会話」として自然に頭に入る。シリーズ多数 (フェルマーの最終定理、ゲーデル、乱択アルゴリズム等)
- **本リポジトリの章**: [`start_here/`](../start_here/README.md), [`00_notation/`](../00_notation/README.md)
- 🔗 https://www.hyuki.com/girl/

### プログラマの数学 第2版
- **著者**: 結城 浩
- **出版**: SB Creative
- 💰 🇯🇵 🌟
- **対象**: プログラミングはできるが数学が苦手な人
- **おすすめ理由**: 数学の概念をプログラミングの観点から、たとえ話豊富に説明。再帰、確率、論理など、ML に役立つ題材揃い
- **本リポジトリの章**: [`00_notation/`](../00_notation/README.md), [`04_discrete_math/`](../04_discrete_math/README.md)

### How Not to Be Wrong
- **著者**: Jordan Ellenberg
- **出版**: Penguin
- 💰 🇬🇧 (邦訳『データを正しく見るための数学的思考』)
- **対象**: 数学的思考に興味がある一般読者
- **おすすめ理由**: 「数学は世界をどう見るか」を物語として伝える名著

## 2. 線形代数 (Linear Algebra)

### プログラミングのための線形代数
- **著者**: 平岡 和幸、堀 玄
- **出版**: オーム社
- 💰 🇯🇵 🌟
- **対象**: プログラマー、エンジニア、ML 学習者
- **おすすめ理由**: 数学者向けの厳密性と、エンジニア向けの実用性を両立。「行列とは何か」をプログラム的に理解できる名著
- **本リポジトリの章**: [`01_linear_algebra/`](../01_linear_algebra/README.md)

### Introduction to Linear Algebra (5th edition)
- **著者**: Gilbert Strang
- **出版**: Wellesley-Cambridge Press
- 💰 🇬🇧 🌟
- **対象**: 大学レベルの線形代数を体系的に学びたい人
- **おすすめ理由**: MIT の伝説的な講義の教科書。直感重視で式の意味が腑に落ちる
- **無料リソース**: 著者の MIT OCW [18.06](https://ocw.mit.edu/courses/18-06-linear-algebra-spring-2010/) で全講義動画が無料公開

### Linear Algebra Done Right
- **著者**: Sheldon Axler
- **出版**: Springer
- 🆓 (PDF が著者サイトで公開) 🇬🇧 🌟
- **対象**: 線形代数を「エレガントに」学びたい人
- **おすすめ理由**: 行列式を最後に持ってくる珍しい構成。固有値・固有ベクトルの本質に最短で到達
- 🔗 https://linear.axler.net/

### 線形代数キャンパス・ゼミ 改訂10
- **著者**: 馬場 敬之
- **出版**: マセマ出版社
- 💰 🇯🇵
- **対象**: 大学初年度生、独学者
- **おすすめ理由**: 図解とステップ計算が豊富で、「教科書では分からない」 を埋める

## 3. 微積分 (Calculus / Analysis)

### やさしく学べる微分積分
- **著者**: 石村 園子
- **出版**: 共立出版
- 💰 🇯🇵
- **対象**: 高校数学 IIIB 程度の知識から、大学初年度生
- **おすすめ理由**: 例題と練習が豊富、計算方法を身につけやすい

### 微分積分学 (高木貞治)
- **著者**: 高木 貞治
- **出版**: 共立出版
- 💰 🇯🇵 🌟 (古典)
- **対象**: 数学科レベル、または「真の理解」を求める人
- **おすすめ理由**: 100 年読まれる名著。$\epsilon$-$\delta$ 論法の厳密性

### Calculus (Stewart)
- **著者**: James Stewart
- **出版**: Cengage
- 💰 🇬🇧 🌟
- **対象**: 大学レベルの標準教科書
- **おすすめ理由**: アメリカで最も使われている微積分教科書。例題・問題が豊富

### 解析入門 I, II
- **著者**: 杉浦 光夫
- **出版**: 東京大学出版会
- 💰 🇯🇵 🌟 (上級)
- **対象**: 数学科の学生、厳密性を求める人
- **おすすめ理由**: 解析学の標準教科書として広く使われる定番

## 4. 確率・統計 (Probability & Statistics)

### 統計学入門 (赤本)

- **出版**: 東京大学出版会
- 💰 🇯🇵 🌟
- **対象**: 統計学の最初の 1 冊
- **おすすめ理由**: 日本の統計学入門書のデファクトスタンダード。「赤本」「青本」「緑本」シリーズの最初
- **本リポジトリの章**: [`03_probability_statistics/`](../03_probability_statistics/README.md)

### データ解析のための統計モデリング入門 (緑本)
- **著者**: 久保 拓弥
- **出版**: 岩波書店
- 💰 🇯🇵 🌟
- **対象**: 統計モデリングを実践したい人
- **おすすめ理由**: 一般化線形モデル (GLM) からベイズまで、実例豊富で読みやすい。「緑本」 として有名

### 現代数理統計学の基礎
- **著者**: 久保川 達也
- **出版**: 共立出版
- 💰 🇯🇵 🌟
- **対象**: 数理統計をしっかり学びたい人
- **おすすめ理由**: 数理統計の標準教科書、定理と証明が体系的

### All of Statistics
- **著者**: Larry Wasserman
- **出版**: Springer
- 💰 🇬🇧 🌟
- **対象**: ML をやる人が統計を一気に学びたいとき
- **おすすめ理由**: 1 冊で確率論から推測統計、ノンパラメトリックまでカバー。コンパクト

### ベイズ推論による機械学習入門
- **著者**: 須山 敦志
- **出版**: 講談社
- 💰 🇯🇵 🌟
- **対象**: ベイズ機械学習に興味がある人
- **おすすめ理由**: ベイズの考え方を ML 文脈で丁寧に。MLP シリーズ屈指の良書

### Causal Inference: What If
- **著者**: Miguel Hernán, James Robins
- 🆓 PDF 公開 🇬🇧 🌟
- **対象**: 因果推論の入門〜中級
- **おすすめ理由**: 因果推論の世界的な教科書、無料公開されている
- 🔗 https://www.hsph.harvard.edu/miguel-hernan/causal-inference-book/

## 6. 機械学習 (Machine Learning)

### ゼロから作るDeep Learning
- **著者**: 斎藤 康毅
- **出版**: O'Reilly Japan
- 💰 🇯🇵 🌟
- **対象**: ML の理論と実装をフレームワーク無しで学びたい人
- **おすすめ理由**: NumPy だけで NN を実装、内部の仕組みが完全に見える。シリーズ全 5 巻
  - 1 巻: 基礎、2 巻: NLP, 3 巻: フレームワーク自作 (DeZero), 4 巻: 強化学習, 5 巻: 自然言語処理 (Transformer)
- **本リポジトリの章**: [`06_ml_math_bridge/`](../06_ml_math_bridge/README.md)

### Pattern Recognition and Machine Learning (PRML)
- **著者**: Christopher Bishop
- **出版**: Springer (2006)
- 💰 🇬🇧/🇯🇵 (邦訳上下) 🌟
- **対象**: ML の理論を体系的に学びたい人
- **おすすめ理由**: ベイズ寄りの ML の世界的バイブル。日本語訳 (上下巻) も評価が高い

### Deep Learning
- **著者**: Ian Goodfellow, Yoshua Bengio, Aaron Courville
- **出版**: MIT Press
- 🆓 PDF 公開 🇬🇧 🌟
- **対象**: 深層学習の基礎から最先端まで
- **おすすめ理由**: 深層学習の標準教科書、線形代数や確率の解説も充実
- 🔗 https://www.deeplearningbook.org/

### Probabilistic Machine Learning (Murphy)
- **著者**: Kevin P. Murphy
- **出版**: MIT Press
- 🆓 PDF 公開 🇬🇧 🌟
- **対象**: ML の最新を確率論観点で学びたい人
- **おすすめ理由**: 2022/2023 出版の「現代版 PRML」。3 巻構成 (Foundations, Inference, Decision Making)
- 🔗 https://probml.github.io/pml-book/

### Hands-On Machine Learning with Scikit-Learn, Keras and TensorFlow
- **著者**: Aurélien Géron
- **出版**: O'Reilly
- 💰 🇬🇧/🇯🇵 (邦訳あり)
- **対象**: ML/DL を手を動かして学びたい実装志向の人
- **おすすめ理由**: 実装中心、Kaggle/業務に直結

### Pythonではじめる機械学習
- **著者**: Andreas Müller, Sarah Guido
- **出版**: O'Reilly Japan
- 💰 🇯🇵
- **対象**: scikit-learn を使った機械学習入門
- **おすすめ理由**: 実用的な ML を Python で学ぶ際の定番

## 📋 「最初の 5 冊」 おすすめセット

予算と相談しながら、以下の 5 冊を順に読むのが万能ルート:

1. **数学ガール** (結城浩) — 数学の世界に親しむ
2. **プログラミングのための線形代数** (平岡・堀) — ML の土台
3. **ゼロから作るDeep Learning** (斎藤康毅) — 実装で理解する
4. **統計学入門 (赤本)** (東大出版会) — 統計の基礎
5. **Pattern Recognition and Machine Learning (PRML)** (Bishop) — ML 理論の到達点

総額 約 25,000 円、所要 1 〜 2 年。これで ML 中級者になれます。
