# 数学コラム生成コマンド

トピック: `$ARGUMENTS`

このトピックについて、`start_here/columns/` にある既存コラムと同じスタイルで、**読み物としてのコラム** を書いてください。

## スタイル指針

### トーン
- 教科書ではなく、雑誌や入門書のエッセイのような口調
- 「〜してみてください」「〜ですよね」などの呼びかけを使う
- 数式は最小限。出すとしても、理由を添えて
- 笑える話・意外な話を入れる

### 構成テンプレート

```
# コラム: <タイトル>

> *<読み始めの一言。お茶でも飲みながら、的な誘い文>*

## はじまりの話
<その概念がどうやって生まれたか、人類の素朴な必要性から>

## 歴史のドラマ
<関連する歴史的人物・事件・誤解・天才のエピソード>

## 言葉の語源
<もし語源に面白い話があれば。algorithm = アル=フワーリズミー、cipher = sifr など>

## 現代の私たちの暮らしへの繋がり
<朝起きてから寝るまでの1日のどこに、それが居るか>

## AI/技術への接点
<もしあれば。なくても省略可>

## 「あれ、これって意外」というポイント
<2〜3個。読み手を「なるほど！」と言わせる>

## 終わり方
<次に何を見ると面白いか、関連コラムへの誘導>
```

## 良いコラムの例

`start_here/columns/` の以下を参考に:
- `00_what_is_math.md` — 数学の5000年史
- `01_pythagoras_story.md` — ピタゴラスの教団のドラマ
- `02_trig_in_real_world.md` — 三角関数の現代利用
- `03_zero_invention.md` — ゼロの発明
- `04_e_and_pi.md` — π と e の話
- `05_math_in_ai.md` — AI と数学

## 注意

- ファクトチェックを意識する: 歴史的人物の生年や事件の年は確からしいものを
- 嘘の逸話を作らない (有名な伝説は「〜と伝えられる」と書く)
- 専門用語が出たら、必ず日常語訳をつける
- 数学アレルギーの人でも最後まで読める長さ・難易度に

## 保存

生成したコラムは、ユーザーの希望に応じて `start_here/columns/` に保存することを提案してください。
