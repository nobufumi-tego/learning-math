# 超ベビーステップ解説コマンド

対象: `$ARGUMENTS`

学習者は **数学に苦手意識があり、記号アレルギーがある初心者** です。
**「数式・専門用語を使わず、たとえ話と日常の例だけで」** 説明してください。

## 厳守ルール

### やること
- たとえ話を3つ以上使う
- 「これって、〜と同じこと」と日常生活に結びつける
- 絵で説明できそうなところは ASCII アートで描く
- 「次に進む前に確認」する問いかけを最後に置く

### やらないこと
- 数式を書かない (`a² + b² = c²` ですらNG、本当に必要なら最後の最後に1行だけ)
- 記号 (∀, ∃, Σ, ∫など) を使わない
- 「自明」「明らか」「簡単」と言わない
- 厳密な定義を出さない
- 専門用語を平気で使わない (出すなら必ず日常語訳をセットで)

## 出力テンプレート

```
# 「<対象>」って、つまり何？

## 一言で
<8〜15文字くらいの、めちゃくちゃ素朴な要約>

## たとえ話
<日常の出来事3つ。料理・買い物・通勤・スマホ操作など、誰でも経験あるもの>

たとえば: <例1>
たとえば: <例2>
たとえば: <例3>

## 何のためにあるの？
<実生活のどこで効いてるか、目に見えるところで>

## 軽く触ってみる
<Python なら数行のコードで体感。数式は使わず、結果がわかるだけ>

## 確認の問いかけ
<読者が自分で言える、「あれ、つまり〜ってこと？」を促す質問>
```

## トーン

- 友達に説明するくらいの口調
- 「〜なんだよね」「〜って気持ちで」など柔らかく
- 笑える例があれば入れる
- 短く区切る (1段落 = 3行以内)
