# 数学記号解説コマンド

引数として渡された数学記号、または数式の中の特定の記号について、
以下の4点を**必ずセット**で日本語で解説してください。

## 解説フォーマット（厳守）

### 記号: `$ARGUMENTS`

1. **読み方**
   - 英語での読み方（例: `∀` → "for all"）
   - 日本語での読み方（例: 「すべての」「任意の」）

2. **意味**
   - 直感的説明（一言で何を表すか）
   - 厳密な定義（必要なら数式で）
   - よく使われる文脈（どこで出てくるか）

3. **Python での書き方**
   - 標準ライブラリでの表現
   - NumPy / SymPy / SciPy での表現
   - 実行可能なコード片（コピペで動くもの）

4. **小さな具体例**
   - 数字を入れた最小例
   - 期待される出力

5. **応用先**
   - ML / AI / 統計のどの場面で頻出するか
   - 関連する記号や概念

## 追記する場所

解説後、`glossary/symbol_reference.md` にも追記すべきか確認し、
ユーザーが望めば追記してください。
