# 03. 三角関数 — スマホ・GPS・音楽・アニメの裏側にいる主役

「サイン・コサイン・タンジェント」と聞くと、高校時代の暗い記憶が蘇る方もいるかもしれません。
でも実は **三角関数は、現代社会で最も使われている数学の一つ** です。

このページを読めば、「**あれもこれも、三角関数だったのか**」という瞬間に何度も出会います。

## まず「直感」だけつかむ

直角三角形を思い浮かべて、**そのうちの1つの角度を「θ（シータ）」** とします。

```
       │\
       │ \
     b │  \  c   (斜辺)
       │   \
       │  θ \
       └─────
          a
```

- **`sin θ` (サイン)** = `b / c` = 「縦 ÷ 斜辺」
- **`cos θ` (コサイン)** = `a / c` = 「横 ÷ 斜辺」
- **`tan θ` (タンジェント)** = `b / a` = 「縦 ÷ 横」（傾き）

要するに、**角度 θ が決まれば、辺の比率も決まる** という関係です。

## 何がうれしいの？

これがなぜ便利なのか、具体例で見てみます。

### 🚁 高い建物の高さを測る

ビルから30メートル離れたところに立って、ビルのてっぺんを**60度の角度で見上げる**。
このビルの高さは？

```
        🏢
        │
        │ h (高さ)
        │
   60°  │
👤──────┘
   30m
```

`tan 60° = 高さ / 30m` なので:
```
高さ = 30 × tan 60° = 30 × 1.732... ≒ 51.96 m
```

メジャーで上まで届かなくても、**角度を測れば高さが分かる**。これが三角測量です。

### 🛰 GPSが位置を特定する仕組み

GPSは、複数の衛星からの距離を測って、地球上のあなたの位置を**三角形の角度関係**から逆算しています。
（実際は球面三角法という発展形ですが、根っこは三角関数）

### 🎵 音楽・音波

音は空気の振動。振動を数式で書くと、まさに **`sin` の波**:

```
音 = A × sin(2π × f × t)
```

- A: 音の大きさ
- f: 周波数（高い音ほど大）
- t: 時間

ラジオ・スマホ通話・Bluetooth、すべてこの波の合成です。

### 🎮 ゲーム・アニメのキャラ移動

キャラクターを円運動させたい?

```python
import math

t = 0.5  # 時間 (秒)
radius = 100
x = radius * math.cos(t)
y = radius * math.sin(t)
# (x, y) が円の上を動く
```

このたった数行で、キャラクターが滑らかに円を描きます。

### 📱 タッチパネルの回転判定

スマホを横向きにすると画面が回転する。あれは加速度センサーの値から、
「重力方向と画面方向のなす角」を `atan2` （タンジェントの逆関数）で計算しています。

### 🤖 機械学習: 画像認識のフィルター

畳み込みニューラルネットワーク(CNN)で使われるフィルターの中には、
**Gabor フィルター**という、`sin` と `cos` の組み合わせで作られたものがあります。
人間の視覚野もこれに近い処理をしていると言われます。

## なぜ「波」になるのか - 円との関係

実は **三角関数は、円の上を回る点の位置** から定義されます。

半径1の円（単位円）を考えて、その上を点が回るとき:

- 横方向の位置 = `cos θ`
- 縦方向の位置 = `sin θ`

時間が経つにつれて θ が増えていくと、点はぐるぐる回って、**横から見ると波**になります。

```
円運動 ←→ 波の動き
これが、振動・音・電波・光のすべてに繋がる「世界の基本パターン」
```

これを発見したことで、人類は電気・無線・コンピュータを作れるようになりました。

## Python で動かしてみる

```python
import math

# 角度は「ラジアン」で渡す（180度 = π ラジアン）
print(math.sin(0))           # 0.0     ← 円のスタート地点
print(math.sin(math.pi / 2)) # 1.0     ← 90度回転
print(math.sin(math.pi))     # ≈ 0     ← 180度（一回転の半分）
print(math.cos(0))           # 1.0     ← 円のスタート（横は1）
print(math.cos(math.pi))     # -1.0    ← 反対側

# 60度の正接
print(math.tan(math.radians(60)))  # 1.732... (= √3)
```

### ラジアンって何？

角度の単位には2種類あります:
- **度 (degree)**: 一周 = 360度（私たちが普段使う）
- **ラジアン (radian)**: 一周 = `2π` ラジアン（数学・プログラミングで使う）

変換: `math.radians(度)` で度→ラジアンに変換できます。
Python の `math.sin` などは**ラジアン**を受け取ることを覚えておいてください。

## まとめ

| 関数 | 三角形での意味 | 実生活での例 |
|---|---|---|
| `sin θ` | 縦 / 斜辺 | 音波、振動、波 |
| `cos θ` | 横 / 斜辺 | 円運動、位相、相関係数 |
| `tan θ` | 縦 / 横 (傾き) | 坂道、視角、画面回転 |

これら3つは、**世の中のあらゆる「周期的な現象」を表す道具**です。
