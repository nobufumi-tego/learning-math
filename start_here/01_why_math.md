# 01. そもそも数学って、何の役に立つの？

「数学なんて、学校のテスト以外で使わないでしょ」と思っている人へ。
**実は今、あなたがこれを読んでいる端末も、数学の塊でできています**。

## 今日のあなたの1日に隠れている数学

朝起きてから寝るまでの、ふつうの1日を追ってみましょう。

### ☀️ 朝

- **スマホのアラーム** が鳴る → 内部の水晶振動子が「1秒間に何回振動するか」を**三角関数の波**として数えている
- 顔認証で画面ロック解除 → カメラが顔の特徴点を捉え、**行列演算**で「あなた」を判定
- 天気予報を見る → 数十兆回の**微分方程式**の計算結果

### 🚃 通勤

- **Googleマップ**で経路を調べる → 道路ネットワークの最短路を**グラフ理論**で計算
- 電車の自動運転 → ブレーキ距離を**微分・積分**で予測
- イヤホンで音楽 → 音は**三角関数の合成**、圧縮は**フーリエ変換**

### 🏢 仕事

- ChatGPT / Claude に質問 → 次のトークンを**確率分布**から選択
- グラフ作成 → **統計**の世界
- ネットショッピングのおすすめ → **行列分解**による推薦

### 🌙 夜

- Netflix のおすすめ → あなたの視聴履歴の**類似度計算**
- ゲームのキャラが滑らかに動く → **三角関数**で円運動を再現
- 寝る前に天体観測アプリ → 地球の自転を**球面三角法**で計算

**つまり、数学を学ぶことは「世界の仕組みを理解する」こと**。
スマホをただ使うのと、「あ、これは三角関数だな」と気付いて使うのとでは、見える世界が違います。

## 数学が「あるパターンの言葉」だと考えるとラク

数学は、**世界を簡潔に書き表すための専用の言語**です。
日本語で「3つあるリンゴと、2つあるリンゴを合わせると5つだね」と言うところを、`3 + 2 = 5` と書く。短くて、誰が読んでも同じ意味になる。

英語が読めると世界中の本を読めるように、数学が読めると**世界中の論文・コード・科学**が読めます。

## 「役に立つ」の3レイヤー

1. **生活で直感を持てる**: 「住宅ローンの利息計算が複雑なのは、指数関数だから」と気付く
2. **仕事の幅が広がる**: データを見て、ただ表を作るのでなく、傾向を読み解ける
3. **AI・科学を理解できる**: 生成AIが何をしているか、どこに限界があるかを判断できる

このリポジトリでは、特に**3番**を目指します。

## 「でも、私は数学が苦手で…」という人へ

学校の数学が苦手だった理由は、ほとんどの場合 **「なぜそれを学ぶか」が説明されなかった** からです。

このリポジトリでは、**常に「なぜ・どこで使うか」をセットで提示**します。
- ピタゴラスの定理 → GPS が距離を計算する仕組み
- 三角関数 → 音楽・スマホ・アニメ
- 微分 → 機械学習の訓練そのもの

「あ、これは知ってると役に立ちそう」と思えるところから、少しずつ進めましょう。
