# 04. 対数 (logarithm) — 地震・音・pH の裏側にいる「桁」の道具

「対数（log）」と聞くと、難しそうですよね。
でも、対数はたった**ひとつの素朴な疑問**から生まれました:

> 「2を何回かけたら1024になる？」

答えは10回（`2¹⁰ = 1024`）。これを `log₂(1024) = 10` と書く。それだけです。

つまり対数は **「何回かけ算したか？」を表す数** です。

## 「桁」を表す道具

対数の真価は、**桁が違いすぎるものを比較しやすくする**ことにあります。

たとえば震度3の地震と震度7の地震。震度は4しか違わないけど、エネルギーは**1000倍以上違う**。
このようにエネルギーが指数的に増減するものを、**人間に分かりやすい数字に圧縮するのが対数**です。

## 実生活で対数が使われる場所

### 🌎 (1) 地震のマグニチュード

マグニチュードは**対数スケール**:
- M5 → M6: エネルギー約**32倍**
- M5 → M7: エネルギー約**1000倍**

マグニチュードが1上がると、エネルギーは約32倍。これは対数の仕組みです。

### 🔊 (2) 音のデシベル (dB)

「ささやき声 = 30dB、地下鉄 = 100dB」と聞いても、3倍くらい？と思いがち。
でも実際は **音のエネルギーは1000万倍違う**。

人間の耳は対数的に音を感じるので、対数の単位「デシベル」が便利。

### 🧪 (3) pH（酸性・アルカリ性）

pH7が中性、pH4は酸性、pH10はアルカリ性。
pHが1違うと、水素イオン濃度が**10倍**違います。これも対数スケール。

### 💾 (4) コンピュータのストレージ

`1 KB = 1024 B`、`1 MB = 1024 KB`、`1 GB = 1024 MB`、`1 TB = 1024 GB`…
**1024 = 2¹⁰** という、2を底とする対数の世界。
「ファイル容量が指数的に増える」のはこのため。

### 📈 (5) 人口・経済成長

「年5%成長」とは、毎年1.05倍。10年で約1.63倍、30年で約4.3倍。
このような**複利的な成長**を見やすくするためにグラフを「対数軸」にすると、直線になります。

### 🤖 (6) 機械学習の損失関数

なんと**機械学習でも対数が大活躍**。
分類問題で頻出する「クロスエントロピー」は:

```
L = − Σ y log(ŷ)
```

ここで `log` が登場します。確率は0〜1の小さい範囲なので、対数を取ると扱いやすくなる。

## 対数の基本ルール

| 数式 | 読み方・意味 |
|---|---|
| `log_a(x)` | a を何回かけたら x になる？ |
| `log₁₀(100) = 2` | 10を2回かけると100 |
| `log₂(8) = 3` | 2を3回かけると8 |
| `ln(e) = 1` | 自然対数（底が `e ≈ 2.718`） |

### 重要な3つの法則

1. **掛け算 → 足し算**: `log(a × b) = log(a) + log(b)`
2. **割り算 → 引き算**: `log(a / b) = log(a) − log(b)`
3. **べき乗 → 掛け算**: `log(aⁿ) = n × log(a)`

これらの法則のおかげで、**コンピュータが無い時代に複雑な計算ができた** という歴史があります。

## Python で動かす

```python
import math

print(math.log10(100))        # 2.0      ← 10² = 100
print(math.log10(1_000_000))  # 6.0      ← 10⁶ = 100万
print(math.log2(1024))        # 10.0     ← 2¹⁰ = 1024
print(math.log(math.e))       # 1.0      ← 自然対数 (底=e)
print(math.log(8, 2))         # 3.0      ← 底を指定（log₂ 8 = 3）
```

### 機械学習でよく見る形

```python
import numpy as np

# 確率 0.9 の対数（クロスエントロピーの中身）
p = 0.9
print(-np.log(p))   # 0.105...   ← 確信があるほど損失は小さい

# 確率 0.01 の対数
p = 0.01
print(-np.log(p))   # 4.605...   ← 外したときの損失は大きい
```

これが「**確信を持って正解 → 損失小、自信満々で外す → 損失大**」という、機械学習の世界の基本原理になっています。

## ネイピア数 e

数学に出てくる謎の数字 **`e ≈ 2.71828...`**。これは:

> 「**1年で100%の利息がもらえる銀行**で、利息を**無限に細かく分けて受け取った**ときの、最終残高の倍率」

具体的には:
- 年1回複利: 2倍
- 月1回複利: 約 2.61倍
- 日1回複利: 約 2.71倍
- 無限に細かく分けて: 約 2.71828...倍 = **e**

なぜこんな変な数が重要なのか？
それは「**自然現象の成長を表す最も自然な数**」だから。
人口・放射性崩壊・電子回路・確率…ありとあらゆる「変化」を表すのに `e` が現れます。
