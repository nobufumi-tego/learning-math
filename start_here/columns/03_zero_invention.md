# コラム 03: ゼロの発明 — 「何もない」を、なぜ書く必要があったのか

> *人類最大の発明は、車輪でも火薬でもなく、「ゼロ」だったかもしれません。*

## 「何もない」を書く必要があった？

「**ゼロ**」と聞くと「当たり前」と思いますよね。
でも、人類が「何もない」を**数として書き表す**ようになったのは、せいぜい1500年前です。

それまでの人類の感覚はこうでした:
- 「羊が3匹いる」 → 数字で書ける
- 「羊が0匹いる」 → なぜ書く必要がある？「いない」と言えばいいじゃん？

人間の素朴な感覚として、「ない」ものを**書く**という発想が、なかったんです。

## ゼロが無いと、どれだけ困るか

ローマ数字を思い出してください。
- 1 = I
- 5 = V
- 10 = X
- 50 = L
- 100 = C
- 1000 = M

これで `2024` を書くと: **MMXXIV**

掛け算してみてください。**MMXXIV × VII** は？
…無理ですよね。**ローマ人は、筆算で計算ができなかった**んです。
複雑な計算はソロバンのような道具に頼っていました。

**位取り (place value) というシステムが無い**ことが致命的でした。
「**32は3が10の位、2が1の位**」みたいな考え方をするには、「位」が空のときに「0」を置く必要があるんです。

## インドで起きた革命

5〜7世紀のインドで、ついに**「ゼロ」が数として扱われる**ようになります。
ブラフマグプタ(598-668年)という天才数学者が、ゼロの計算ルールを体系化しました:

- `x + 0 = x`
- `x − x = 0`
- `x × 0 = 0`

そして「**0 ÷ 0 は定義できない**」という、現代まで続く慎重さも示しました。

インド人は「**シューニャ (śūnya)**」（=空）という言葉でゼロを呼びました。
これは仏教の「空（くう）」と同じ語源。**「無の概念を哲学で深く考えていた文化**」から、ゼロが生まれたのは偶然じゃないかもしれません。

## アラビアを経て、ヨーロッパへ

インドのゼロは、アラビア商人を通じてイスラム世界に伝わりました。
アラビア語では **「シフル (sifr)」**。

そしてイスラム世界からヨーロッパに伝わったのが12〜13世紀。
イタリアの数学者**フィボナッチ**（あの「フィボナッチ数列」の人）が、北アフリカで習った数字をヨーロッパに紹介する本『**Liber Abaci（算術の書）**』を書きました(1202年)。

ところがヨーロッパはこれを**警戒**します。
- 「ゼロは悪魔の数字」
- 「アラビア数字は偽造しやすい」（実際、0を6に書き直すなどの偽造があった）

フィレンツェでは1299年、**「商業帳簿でアラビア数字を使うことを禁止する」** 法律が作られたほど。
人類は「ゼロ」を受け入れるのに、500年もかかったんです。

## 言葉の中に残る「シフル」

「シフル (sifr)」は英語に入って「**cipher (サイファー)**」になりました。
意味は「ゼロ」「暗号」「謎の人物」。

そして「シフル」のフランス語版が「**chiffre**」、これがさらに変化して **「数字 (digit)」**全般を指す言葉に。
最終的に英語の **"zero"** はイタリア語の `zero`、それも `zefiro` → アラビア語 `sifr` 由来。

つまり、「ゼロ (zero)」「シファー (cipher)」「ディジット (digit)」、これらは**全部同じ単語**から枝分かれしたものです。
**インド→アラビア→イタリア→ヨーロッパ全域**という旅路の証拠が、言葉の中に残っています。

## ゼロがあると何ができるようになったか

(1) **位取り** が成立し、筆算ができるようになった
(2) **負の数** が考えられるようになった（ゼロより小さい数）
(3) **デカルト座標** が成立した（原点 = ゼロ）
(4) **微積分** の基礎ができた（極限 → ゼロに近づく）
(5) **2進数** が可能になった（0と1の世界 = コンピュータ）

つまり、**コンピュータ・現代の科学・経済**、すべてはゼロの発明のおかげです。

## あなたの財布の中のゼロ

クレジットカード番号、商品のバーコード、IPアドレス、銀行の口座番号…
**現代社会のあらゆる数字に「0」が含まれています**。

財布から1000円札を出してみてください。
そこに書かれている「1000」という数字を見て、「**この0が無かったら、人類は今もローマ数字で MMM と書いていたな**」と考えると、ちょっと感動しませんか？

## 余談: なぜインドだったのか

なぜゼロが「インド」で発明されたのか、には諸説あります。

- 仏教・ヒンドゥー教の「無」「空」の哲学が深かった
- サンスクリット語の文法が、数学的に厳密だった
- 天文学・占星術の必要性
- 大きな数（**ナユタ・恒河沙・阿僧祇**など仏教用語）を扱う文化があった

「**0**」を発見できる文化的土壌があった、ということです。

数字一つの背後に、こんなドラマがあります。
次にあなたが何気なく `0` をタイプするとき、5世紀のインドの空気を、少しだけ思い出してください。
