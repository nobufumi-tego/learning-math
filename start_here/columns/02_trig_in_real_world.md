# コラム 02: 三角関数は、現代社会のどこで暮らしているか

> *「サイン、コサイン、いつ使うの？」と聞いた高校生のあなたへ。*

## 結論: ほぼ毎日、あなたの周りで動いています

三角関数は **「周期的に繰り返すもの」を扱う最強の道具** です。
そして現代社会には、周期的なものが溢れています。

## 朝起きてから寝るまでの三角関数

### 🛏 6:00 アラームが鳴る
**スマホ内の水晶発振器** が、決まった周波数で振動しています。
振動はまさに `sin` の波。1秒間に何回振動したかをカウントして時刻を進めています。

### 🚿 6:30 シャワー
お湯と水を混ぜるバルブの裏には、水の流れの**乱流**があります。
乱流の解析には**フーリエ変換**（三角関数の合成）が使われます。

### 📱 7:00 SNS をチェック
スマホの画面に表示される画像は、**JPEG**形式で圧縮されています。
JPEG の心臓部は **離散コサイン変換 (DCT)**。文字通りコサインです。

### 🎵 7:30 通勤中、音楽を聴く
イヤホンから聞こえる音は、空気の波 = `sin` の波。
Bluetooth で送られた音声データは、波を**フーリエ変換で周波数に分解→圧縮→送信→再合成**しています。

### 🛰 8:00 Googleマップで道を調べる
GPS は3つ以上の衛星からの距離を測って、あなたの位置を**三角測量**で特定。
地球の丸さを考慮した**球面三角法**が動いています。

### 🏢 9:00 オフィスで建物のCADを見る
ビルや橋の設計は、**有限要素法**で力学計算されています。
振動を扱うとき、必ず `sin` と `cos` の組み合わせが現れます。
**地震が来てもビルが倒れない**のは、三角関数を使った計算のおかげ。

### 🛜 10:00 Zoom 会議
画面に映る映像は、**動画コーデック（H.264/H.265）** で圧縮されています。
これも DCT（離散コサイン変換）を使う。
**1秒あたり数百MBのデータが、数MBに圧縮されてリアルタイム送受信できる**のは、三角関数のおかげ。

### 🎮 12:00 昼休み、ゲームで遊ぶ
3Dゲームのキャラクターを動かすのは「**回転行列**」。
中身は `sin` と `cos`。
```
x' = cos(θ) × x − sin(θ) × y
y' = sin(θ) × x + cos(θ) × y
```
これがゲーム内のあらゆる回転処理を支えています。

### 🚀 14:00 ロケット打ち上げのニュース
ロケットの軌道計算、姿勢制御。
すべて**三角関数 + 微分方程式**の世界。

### 📷 17:00 帰り道、写真を撮る
スマホのカメラの**手ぶれ補正**は、加速度センサーが検知した揺れを `atan2` で角度に変換し、レンズを動かしています。

### 🎧 19:00 家に着いて Spotify を起動
楽曲推薦の裏側でも数学が動いていますが、音そのものが既に `sin` の合成。
バッハのフーガもKendrick LamarのHipHopも、本質は**周波数の組み合わせ**。

### 🔬 21:00 ドラマで医療シーンを見る
MRI（磁気共鳴画像装置）は、人体内の水素原子の**回転**（核磁気共鳴）を撮影しています。
これも完全に三角関数の世界。
**癌の早期発見ができる**のは、三角関数のおかげと言ってもいい。

### 💤 23:00 寝る
寝ている間、家電の電力消費はぐらぐら変動しますが、電力会社は **交流電流の波**（`sin` の波）として制御。
交流電流を発見・実用化したテスラ vs エジソンの「**電流戦争**」は、三角関数を巡る戦いでもありました。

## 三角関数の「四大利用カテゴリ」

(1) **周期現象** — 音、光、電波、振動、波
(2) **回転** — ゲーム、ロボット、CG、衛星
(3) **座標変換** — 角度↔長さ、極座標↔直交座標
(4) **信号圧縮** — JPEG、MP3、動画、通信

## 三角関数と AI の意外な接点

### 1. Positional Encoding (Transformer)
ChatGPT/Claude の中核技術 **Transformer** は、文の中の単語の位置情報を **`sin` と `cos` の組み合わせ** で埋め込みます:

```
PE(pos, 2i)   = sin(pos / 10000^(2i/d))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d))
```

つまり、**ChatGPTが「言葉の順番」を理解するのに、三角関数を使っている**わけです。

### 2. 画像認識のフィルター
畳み込みネットワーク (CNN) では、**Gabor フィルター**という三角関数ベースのフィルターが使われることがあります。
これは人間の脳の視覚野の働きに近い、と言われています。

### 3. 音声認識・音楽生成
音声を扱うあらゆる AI は、入力をまず**スペクトログラム**（時間 × 周波数）に変換します。
これは三角関数（フーリエ変換）の出力。

## 「あの式、たぶんサインだな」と気付けるようになる

数学が「日常」になる瞬間は、こういう気付きです。

- Wi-Fi のアンテナ → 電波 → sin の波
- カラオケのエコー → 信号処理 → フーリエ変換
- 心電図 → 周期信号 → 三角関数
- 株価のチャート → 周期成分の分析 → フーリエ
- 心臓のドキドキ → 周期 → 振幅 sin
- 月のリズム → 潮汐 → 三角関数

**世界は、思った以上に「波」でできています**。
