# コラム 04: π と e — 自然界の「2大スター数」の不思議

> *数学の世界には、何度繰り返しても枯渇しない、永遠の数があります。*

## π (パイ) ≈ 3.14159265358979...

円の周りの長さは、直径の何倍か？
答えは **`π` 倍**。

```
円周 = π × 直径
```

これだけのことなのに、`π` は不思議な数です。
**整数の比で表せない (無理数)、しかも代数方程式の解にもなれない (超越数)**。
小数点以下は永遠に続き、規則的なパターンも見つかっていません。

### π はどう測られたか
- 紀元前2000年のバビロニア: `π ≈ 3.125`
- アルキメデス (紀元前250年頃): 96角形を使って `3.1408 < π < 3.1428`
- 中国の祖沖之 (5世紀): `π ≈ 3.1415926` （7桁正確）
- 現代: 兆桁まで計算済み（実用的な意味はないが、計算機性能の指標になる）

### なぜ無理数なのか
驚くべきことに、円という幾何学的に「シンプル」な形を表す数が、**整数の比では絶対に表せない**。
これは数学の深いミステリーの一つで、証明 (1761年、ランベルト)に到るまで2000年以上かかりました。

### π が出てくる意外な場所

円とは関係なさそうな式にも、`π` は突然出てきます:

- **正規分布の確率密度関数**: `f(x) = (1/√(2π)) × exp(-x²/2)`
- **不確定性原理**: `Δx × Δp ≥ ℏ/2`（量子力学）
- **オイラーの公式**: `e^(iπ) + 1 = 0` （後述）

つまり、**「波」「確率」「物理」のあちこちに π が顔を出す**。
円や三角関数を経由して、自然界のあらゆる現象に関わっているからです。

## e (ネイピア数) ≈ 2.71828182845904...

もう一つの主役 **`e`**。これも超越数で、永遠に続く小数。

### e はどこから来たか
ジョン・ネイピア (16世紀スコットランド) が**対数表**を計算するときに気付き、後にオイラーが詳しく研究しました。

最も直感的な定義は **「複利」** から:

> 「**1年で元本が100%増える銀行**で、その利息を**無限に細かく分けて再投資**したら、年末にいくらになる？」

- 年1回複利: `(1 + 1/1)¹` = **2倍**
- 月1回複利: `(1 + 1/12)¹²` ≈ **2.613**
- 日1回複利: `(1 + 1/365)³⁶⁵` ≈ **2.715**
- 1秒ごと、いやもっと細かく → 究極の限界は **`e` ≈ 2.71828...**

つまり `e` は「**連続的な成長の天井**」を表す数です。

### e はなぜ自然界に現れるのか

「**変化の速度が、現在の量に比例する**」ような現象は、すべて `e` で記述されます:

| 現象 | 内容 |
|---|---|
| 細菌の増殖 | 個体数が多いほど、増える数も多い |
| 放射性物質の崩壊 | 残っている量に比例して減る |
| 銀行の複利 | 元本が多いほど利息も多い |
| 電子回路の充電・放電 | 電圧差に比例して電流が流れる |
| 化学反応 | 濃度が高いほど反応が速い |
| 人口 | 多いほど増える（資源無視の理想化） |

「**増減速度 = 現在の値**」という関係を満たす関数は、ただ一つ **`f(x) = eˣ`** だけ。
これが「自然対数」と呼ばれる所以です。

### 機械学習での e

機械学習でも `e` は大活躍:

- **シグモイド関数**: `σ(x) = 1 / (1 + e⁻ˣ)` — ニューラルネットの活性化関数
- **ソフトマックス**: `softmax(zᵢ) = e^zᵢ / Σ e^zⱼ` — 分類問題の出力層
- **正規分布**: `exp(-x²/2)` — 確率分布の王様

つまり、ChatGPT が次の単語を選ぶときの「確率分布」を作るのに、`e` が使われています。

## π と e は、本当に独立?

二つの主役 `π` と `e` は、見るからに関係なさそうですよね。
- `π` は円から来た
- `e` は複利から来た

ところが、18世紀の**レオンハルト・オイラー**が、世界を揺るがす式を発見します:

```
   e^(iπ) + 1 = 0
```

これが「**オイラーの等式**」、史上もっとも美しい式と呼ばれるもの。

なんとこの一つの式の中に:
- **`e`** （自然対数の底）
- **`i`** （虚数単位 = √(-1)）
- **`π`** （円周率）
- **`1`** （乗法の単位元）
- **`0`** （加法の単位元）

数学界の**主要メンバー全員**が登場し、しかも「`e^(iπ) = −1`」というシンプルな関係を満たしている。
**人類最大の知的成果のひとつ**と言われます。

リチャード・ファインマン（物理学者）はこの式を「**数学のもっとも素晴らしい式**」と評しました。
高校生のとき、これを知って物理に進んだ、と語る科学者は多いです。

## π と e は、私たちの一部

- スマホで写真を撮るとき → 光のレンズ計算に **`π`**
- 銀行の住宅ローン金利 → 複利計算に **`e`**
- 心拍数の解析 → 信号処理で **`π`**
- LINE で送るメッセージ → 暗号化アルゴリズムに **`e`**
- AI が画像を分類するとき → ソフトマックスで **`e`**

私たちの日常は、**π と e に支えられて動いています**。

## 試してみる

```python
import math

print(math.pi)         # 3.141592653589793
print(math.e)          # 2.718281828459045
print(math.exp(1))     # 2.718281828459045  ← e の別の書き方

# 複利で e に近づく実験
n = 1
print((1 + 1/n) ** n)         # 2.0
n = 12
print((1 + 1/n) ** n)         # 2.613...
n = 1_000_000
print((1 + 1/n) ** n)         # 2.7182804...

# オイラーの等式
import cmath
result = cmath.exp(1j * math.pi) + 1
print(result)                  # 1.2e-16 + 0j  (浮動小数点誤差で完全な0にはならない)
```

最後の結果が **ほぼ 0** になるのを見たとき、人類の知的偉業をちょっとだけ実感できます。

## まとめ

| 数 | 値 | 出自 | 現代での使われ場所 |
|---|---|---|---|
| `π` | 3.14159... | 円周 / 直径 | 波・物理・確率・グラフィックス |
| `e` | 2.71828... | 連続複利 | 自然現象・統計・機械学習 |

そして、この2つを `i`（虚数）で繋ぐと、`e^(iπ) + 1 = 0` という詩のような式が現れる。

数学は単なる計算じゃなくて、**詩のような世界** でもあります。
