## シェルと Python の関係

ここまでは **シェル** (bash, zsh, PowerShell) という言語を覚えてきました。
シェルは「**ファイル操作と命令の連携**」が得意。

でも、シェルが苦手なこともあります:
- 複雑な計算
- 表データの分析
- グラフを描く
- 機械学習やAI

そういう仕事を任せられる、もっと賢い言葉が **Python (パイソン)** です。

シェルとPython の関係は:
- **シェル** = 家事全般をこなす器用な家政婦
- **Python** = ちゃんと数学も話せる、頭のいい家庭教師

両方覚えておくと最強。
そしてターミナルは、**両方を呼び出せる玄関口**です。

## Python の歴史 (1分で)

Python は **1991年**に **オランダのプログラマー Guido van Rossum (グイド・ヴァンロッサム)** が作りました。
Guido さんはイギリスのコメディ番組「**Monty Python's Flying Circus**」のファンで、そこから名前を取りました。

**ヘビ (python) と関係ない**んです、本当は。
でもロゴはヘビになっています (たぶん「分かりやすいから」)。

Python は「**読みやすさ**」を最優先に設計されていて:
- 英語の文章みたいに読める
- インデント (字下げ) で構造を表す
- 短く書ける

そのおかげで、今や **世界一人気のあるプログラミング言語**(2026年時点) になりました。
特に **AI / 機械学習** の世界では、ほぼ Python が標準です。

## ペンタに Python を呼んでもらう

ターミナルで:

```bash
python3
```

(Mac/Linux) または

```powershell
python
```

(Windows) と打つと、ペンタの隣に **Python さん**が現れます。
こんな表示:

```
Python 3.11.0 (main, Oct 24 2022, 16:07:38)
[Clang 14.0.0] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>>
```

`>>>` が、**Python の "プロンプト"**。シェルの `$` と同じく、「命令待ってます」のサイン。

### 試してみる

```python
>>> 1 + 1
2

>>> print("こんにちは Python")
こんにちは Python

>>> 100 * 100
10000

>>> 2 ** 10           # 2を10乗 (2¹⁰)
1024
```

**Python は、超高機能な電卓**でもあります。

抜けるには:
```python
>>> exit()
```

または `Ctrl+D` (Mac/Linux) / `Ctrl+Z + Enter` (Windows)。

## ファイルに書いて実行する

毎回 `python3` を起動して打つのは面倒。
普通は **ファイルに書いて、ファイルを実行**します。

```bash
# テキストエディタで hello.py を作る
echo 'print("こんにちは、Python")' > hello.py

# 実行
python3 hello.py
```

返事:
```
こんにちは、Python
```

## ちょっと意味のある例: 100まで足す

ターミナルで電卓を叩くより、Python に頼んだ方が速い:

```python
>>> sum(range(1, 101))
5050
```

「1 から 100 までの和」が、たった一行で。
小学校の算数で出てくる「1 + 2 + 3 + ... + 100 = 5050」(ガウスの小学生時代の逸話) を、Python は瞬殺します。

## リスト・辞書 — Python の道具箱

Python には**データを整理する箱**がいくつかあります:

### リスト (順序つきの並び)
```python
>>> fruits = ["りんご", "バナナ", "みかん"]
>>> fruits[0]
'りんご'
>>> len(fruits)
3
```

### 辞書 (キーと値のペア)
```python
>>> ages = {"太郎": 25, "花子": 30}
>>> ages["太郎"]
25
```

これらの「箱」を組み合わせて、複雑なデータを扱えるようになります。
**機械学習も、結局は箱の中の数字を整理する作業**です。

## NumPy — 数学のための道具

Python には、外部の道具 (**ライブラリ**) を組み込んで使う仕組みがあります。
**NumPy (ナンパイ)** は、数学・科学計算のための定番ライブラリ:

```python
>>> import numpy as np
>>> x = np.array([1, 2, 3, 4, 5])
>>> x * 2
array([ 2,  4,  6,  8, 10])
>>> np.mean(x)
3.0
```

`x * 2` で「**全要素を2倍**」が一発。
これが**機械学習の基本演算**そのものです。
