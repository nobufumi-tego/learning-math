# 02. ペンタに挨拶する — 「シェル」って何者？

```
       ／￣￣＼
      |  ・ω・  |   やっと会えたね。
      |   ⌒    |   ぼくが、きみのターミナルに住んでるペンタだよ。
       ＼____／
```

## ペンタの正体: シェル (Shell)

ターミナルを開くと、こんな画面が出ます:

```
nobufumi@laptop ~ $ █
```

この **`$`** の前で待ってる存在、それが **シェル (shell)**。
このリポジトリでは、シェルを **ペンタ** と呼びます。

シェルの仕事は3つ:

1. あなたが打った命令を**読む**
2. それを OS が分かる形に**翻訳する**
3. 結果をあなたに**見せる**

つまり**通訳**です。日本人 (あなた) と OS (英語しか分からない外国人) の間に立って、お互いを繋いでくれます。

## なぜ「シェル (殻)」と呼ぶのか

シェル = 「貝殻、外殻」の意味。

```
[あなた] ⇄ [シェル (殻)] ⇄ [カーネル (核)] ⇄ [ハードウェア]
```

OS の中心部分は **カーネル (kernel = 核)** と呼ばれていて、そこは怖くて触れません。
シェルは、そのカーネルを**包む殻** として、外から触れる安全な層を提供してくれている。

つまり、ペンタは **「安全に OS の力を引き出すための入口」** です。

## シェルにはいろんな種類がある

ペットに犬・猫・鳥がいるように、シェルにも品種があります:

| 名前 | どこで使う | 特徴 |
|---|---|---|
| **bash** (Bourne Again Shell) | Linux のデフォルト、Macも昔そうだった | 一番有名、世界中で使われている |
| **zsh** (Z Shell) | macOS の現在のデフォルト | bash の上位互換、機能豊富 |
| **fish** | 上級者・好き者 | 補完がめちゃくちゃ賢い |
| **PowerShell** | Windows のデフォルト (新しいやつ) | Microsoft が作った、パワフル |
| **cmd** (コマンドプロンプト) | Windows の昔から | レガシー、もう使うこと少ない |

このリポジトリの例は、bash / zsh / PowerShell すべてで動きます。

## あなたのペンタは何?

ターミナルを開いて、こう打ってみてください:

```bash
echo $SHELL
```

(Mac/Linux の場合)

返事:
```
/bin/zsh        ← zsh ペンタ
/bin/bash       ← bash ペンタ
/usr/bin/fish   ← fish ペンタ
```

Windows の PowerShell の場合:
```powershell
$PSVersionTable.PSVersion
```

これで自分のペンタの名前が分かります。

## プロンプト — ペンタの「待ってます」サイン

ターミナルの最後に出ている、こういうやつ:

```
nobufumi@laptop ~ $
```

これを **プロンプト (prompt)** と呼びます。意味は「促す、催促する」。
**「ペンタが、あなたの命令を待ってるよ」** というサインです。

それぞれの部分:

| 部分 | 意味 |
|---|---|
| `nobufumi` | あなたのユーザー名 |
| `@` | 区切り (アットマーク) |
| `laptop` | コンピュータの名前 (ホスト名) |
| `~` | 今いる場所 (詳しくは次ページ) |
| `$` | 「ふつうのユーザー」のサイン (`#` だと管理者) |

## 命令を打ってみる: 一番安全な命令

ペンタにとって**一番無害**な命令を打ってみましょう。

```bash
echo こんにちは
```

返事:
```
こんにちは
```

`echo` は「**おうむ返し**」する命令。
「こんにちは」とペンタに言ったら、ペンタも「こんにちは」と返してくれた、ただそれだけ。

でもこれ、人類の最初のコンピュータ対話の感触と、ほぼ同じです。
1970年代の人々も、こうやって VT100 で `HELLO WORLD` と打って感動していました。

## ペンタは「あなたが今いる場所」を覚えている

ペンタは、いつでも**「今、自分がコンピュータのどこにいるか」**を意識しています。
ペットの犬が「家のどの部屋にいるか」を分かっているのと同じ。

これを **カレントディレクトリ (current directory)** と言います。
日本語だと「**いまの作業場所**」くらいの意味。

ペンタに「今どこ？」と聞くには、次のページで紹介する `pwd` を使います。

## まとめ

- **シェル**は、あなたと OS の間の通訳
- このリポジトリでは、シェルを **ペンタ** と呼ぶ
- ペットに種類があるように、シェルにも種類がある (bash, zsh, fish, PowerShell)
- `$` の前 = ペンタが命令を待ってる状態 (プロンプト)
- ペンタは「今いる場所 (カレントディレクトリ)」を意識している
- 一番安全な命令は `echo`

## 🔍 ググってみよう

- **bash** — 1989年から使われている、世界で一番有名なシェル
- **zsh** — bash より機能豊富。Mac の標準
- **oh-my-zsh** — zsh をオシャレにするカスタマイズフレームワーク
- **PowerShell** — Microsoft が作った、オブジェクト指向のシェル
- **Bourne shell** — bash の祖先 (1979年)
- **シェルスクリプト** — シェルの命令をファイルにまとめたもの。自動化の第一歩
- **dotfiles** — `.bashrc` `.zshrc` などの設定ファイル群。プロは大事に育てる
- **starship** — モダンでカスタマイズ自由なプロンプト
