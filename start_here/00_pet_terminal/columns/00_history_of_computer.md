# コラム 00: コンピュータ80年史 — 真空管からスマホまで

> *疲れたら、お茶でも飲みながら。今日は、あなたが今触っている「箱」がどこから来たかの話です。*

## 1940年代: 部屋いっぱいのコンピュータ

最初の本格的なコンピュータは、**1946年のENIAC** (エニアック)。
アメリカで作られた、**部屋いっぱいの真空管の塊**でした。

スペック:
- 重さ: **30 トン**
- 真空管: 17,468本
- 消費電力: 150kW (家30軒分)
- 性能: 1秒間に5,000回の足し算

ちなみに、あなたのスマホは ENIAC の **数十億倍** 速いです。
**80年で数十億倍**。これがコンピュータの進化のすごさ。

ENIAC の設計者の一人 **ジョン・フォン・ノイマン** は、後に「**ノイマン型**」というコンピュータの基本構造を考案します。
今のスマホもパソコンも、結局はノイマン型です。**80年前の設計が現役**。

## 1950年代: トランジスタ登場

真空管はデカくて、すぐ切れるし、熱を出す。
**1947年に発明された "トランジスタ"** が、これを置き換え始めます。

トランジスタは小さくて、長持ちで、熱が少ない。
コンピュータが**大きな冷蔵庫サイズ**に収まるように。

発明者の一人 **ウィリアム・ショックレー** は、後にシリコンバレーの父と呼ばれます。
彼の研究室から飛び出した若者たちが、Intel や AMD を作りました。

## 1960年代: 大型機 (メインフレーム) の時代

**IBM** が圧倒的な存在感を発揮した時代。
銀行・政府・大企業に「**System/360**」という大型コンピュータが導入されます。

この頃、**プログラミング言語** の発明が相次ぎます:
- **Fortran** (1957) — 科学計算
- **COBOL** (1959) — 事務計算 (今も銀行システムで現役)
- **Lisp** (1958) — AI研究の祖
- **C 言語** (1972) — OS を書くための言語

そして1969年、AT&T ベル研究所で **ケン・トンプソン** と **デニス・リッチー** が、後の世界を変える OS を書き始めます。

その名も **UNIX** (ユニックス)。

## 1970年代: UNIX とパーソナルコンピュータの始まり

UNIX は最初、ベル研究所の社内ツールでした。
でも「**シンプルで美しい**」設計が、世界中の研究者を魅了します。

UNIX で生まれた文化:
- パイプ `|`
- リダイレクト `>`
- 「小さく作って組み合わせる」哲学
- C 言語

これらは今でも、**Linux・macOS・Android・iOS の根っこ**で生きています。

そして1975年、若者2人が **「マイコン用の言語」** を売り出します。
**ビル・ゲイツ** と **ポール・アレン**、Microsoft の創業です。

1976年には、ガレージから **Apple** が誕生 (スティーブ・ジョブズ&ウォズニアック)。

## 1980年代: PC とグラフィカル UI

1981年、**IBM が IBM PC** を発売。
これにより「個人で買えるコンピュータ」が一般化します。

1984年、**Apple Macintosh** が登場。
史上初の本格的な**マウス + ウィンドウ + アイコン** UI を一般化させた製品。
Macintosh の有名な広告 (Super Bowl の "1984") は今でも語り継がれます。

1985年、**Microsoft Windows 1.0** リリース。
こうして「**マウスでクリックする**」操作が世界に普及しました。

ところがこの裏で、ターミナル文化は**死なずに進化を続けた**。
プロのエンジニア・科学者・サーバー管理者は、ずっとターミナルで仕事をしていました。

## 1990年代: インターネット革命

1989年、**ティム・バーナーズ＝リー** が CERN で **WWW** (World Wide Web) を発明。
1991年、**フィンランドの大学生 リーナス・トーバルズ** が、趣味で OS を作り始めます。
それが **Linux**。

リーナスは「ペンギンが好きだから」という理由で、Linux のマスコットを **ペンギン (Tux)** にしました。
このリポジトリの **ペンタ** も、Tux への敬意から名付けられました。

```
       ／￣￣＼
      |  ・ω・  |   ぼくの先輩、Tux のおかげで生まれたよ！
       ＼____／
```

1995年、**Windows 95** がメガヒット。
1996年、**Google** がスタンフォード大の大学院生プロジェクトとして始動。
2001年、**Mac OS X** 登場 (UNIX 系)。
2007年、**iPhone** 発表。

## 2000〜2010年代: クラウドとモバイル

2006年、**Amazon が AWS** を立ち上げ。**クラウドコンピューティング** 時代の幕開け。
2008年、**Android** 登場。
2010年、**Docker** 登場 — コンテナ革命。

サーバーは**地球の裏側**に置かれるようになり、CLI で遠隔操作するスキルが**必須**になります。
これも、ターミナル文化が**逆に重要性を増した**理由のひとつ。

## 2020年代: AI 革命

2020年、**GPT-3** が登場。
2022年、**ChatGPT** が一般公開され、世界が震撼。
2023年、**Claude**、**Gemini** が相次いで発表。
2024年、**Claude Code**、**Gemini CLI**、**Codex CLI** など、**AI が CLI に統合された世代**が登場。

つまり今、私たちは:

```
  ENIAC (1946)
     ↓ (80年)
  AI と日本語で会話するターミナル (2026)
```

という、人類史的な変化の途中にいます。

## あなたのスマホ vs 全人類の昔の計算力

ちょっとびっくりする事実:
- **iPhone 15 の計算性能**: 約 35 TFLOPS (兆回/秒)
- **1985年時点での全世界のコンピュータの合計計算性能**: それ未満

つまりあなたのスマホ1台で、**1985年の全人類のコンピュータより速い**。
それを片手で持って、SNS見ながら通勤してる現代人、すごい時代に生きてます。

## 進化の本質: 道具がだんだん「人に近づく」

歴史を振り返ると、コンピュータは**人間に近づき続けて**います:

| 時代 | 操作方法 | 抽象度 |
|---|---|---|
| 1940年代 | スイッチを物理的にON/OFF | 機械語 |
| 1950年代 | パンチカードに穴 | アセンブラ |
| 1960年代 | テレタイプで英語コマンド | 高級言語 (Fortran) |
| 1980年代 | マウスでクリック | GUI |
| 2000年代 | スマホをタップ | タッチUI |
| 2020年代 | **日本語で話す** | 自然言語 |

そして次は…?
**人間の言葉でコンピュータと会話する**のが、もう普通になってきています。
これがこのリポジトリで学ぶ世界です。
