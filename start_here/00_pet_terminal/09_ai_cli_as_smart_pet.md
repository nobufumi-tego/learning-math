## CLI の歴史的瞬間

ターミナル CLI は **1970年代から半世紀以上**、本質的にほぼ変わりませんでした:

- 命令を**正確に覚えて**
- **一字一句正しく**打つ
- 間違えるとペンタは**何もしてくれない** (`command not found`)

これが2024年以降、**根本的に変わりました**。

**AI CLI** の登場で、こうなったのです:

```bash
$ ぼく日本語で「昨日のログから ERROR を含む行を取り出して、件数を月別に集計して」って言える？

[AI] もちろんできます。以下のコマンドで実行できます:
     awk -F' ' '/ERROR/{print $1}' yesterday.log | cut -d'-' -f2 | sort | uniq -c
```

**自然言語 → CLI 命令** が一瞬で。
昔の SF 映画みたいな世界が、**もう実現してしまった**のです。

## 3つの主要 AI CLI

### 1. Claude Code (Anthropic)

```bash
# インストール (Node.js が必要)
npm install -g @anthropic-ai/claude-code

# 起動
claude
```

Anthropic 社が作った CLI。**コードを書く・読む・直す**ことに特化していて、世界中のエンジニアが使っています。
このリポジトリも、Claude Code のために最適化されています。

### 2. Gemini CLI (Google)

```bash
# インストール手順は公式参照
gemini
```

Google の Gemini AI を CLI で使う。
画像認識など Google の技術が使えるのが強み。

### 3. Codex CLI (OpenAI)

```bash
# インストール手順は公式参照
codex
```

ChatGPT を作った OpenAI 社の CLI。
「コードを書く」 ことに古くから取り組んでいる老舗。

## 何が革命的なのか

### 革命 1: コマンドを暗記する必要がなくなった

昔:
- `ls` `cd` `grep` `awk` `sed` `xargs` ... を全部覚える
- オプション (`-rf` `-i` `-l` ...) も覚える
- それでも忘れて Stack Overflow に何度も同じ質問

今:
- 「やりたいこと」を**日本語で言う**だけ
- AI が適切なコマンドを提案
- 自分は「**実行するか確認**」するだけ

### 革命 2: コードを書ける人の幅が広がった

「プログラミングできる人 = 専門教育を受けた人」だった時代から、
「**やりたいことが言葉にできる人 = 全員**」が、コードで何かを作れる時代に。

医師・弁護士・主婦・小学生・年配の方も、AI CLI を通じて自動化や分析ができます。
**これは産業革命に匹敵するインパクト**だと言われます。

### 革命 3: 学習速度が劇的に上がる

「ググって、Stack Overflow で答えを探して、自分の問題に合わせて改造する」が、
「**AI に質問して、自分に合った答えを最短で得る**」になりました。

このリポジトリの学習も、CLI から AI に質問しながら進められます。

## ペンタと AI CLI の関係

```
[あなた]
    ↓ (日本語)
[AI CLI (Claude/Gemini/Codex)]   ← ペンタの兄貴分
    ↓ (シェルコマンド)
[ペンタ (シェル)]
    ↓ (システムコール)
[OS / カーネル]
    ↓
[コンピュータ]
```

つまり、AI CLI はペンタを**指揮する指揮者**。
あなたは指揮者に「こんな曲を演奏したい」と日本語で伝え、ペンタが**実際に演奏**してくれる。

## AI CLI の強みと限界

### 強み
- **試行錯誤が爆速**: 思いついたアイデアを即試せる
- **学習しながら使える**: 動かしながら理解できる
- **エラーが怖くない**: エラーメッセージをそのまま AI に貼ればいい
- **言語の壁がない**: 日本語でも英語でも、何語でもOK

### 限界
- **間違いもある**: AIは時々嘘をつく (ハルシネーション)
- **ブラックボックス化**: 「なぜそれで動くか」を理解せずに進むと、後で困る
- **責任は人間**: AI が書いたコードでも、本番に出す前に**人間が確認**

このリポジトリは、AI に任せきりではなく、**自分でも理解する** ための道しるべです。
AI を使いつつ、**裏側の数学とプログラミングも理解する**。
これが、**AI を使いこなす人**と**AI に使われる人**の分かれ道です。

## 学習を始めるための、おすすめの「最初の質問」

ターミナルで `claude` を起動して、こう聞いてみてください:

> このリポジトリ全体の構成を教えて

> 私は数学が苦手で、ターミナルもさっき覚えたばかり。今日、何から始めるのがおすすめ?

AI が、あなたのレベルに合わせて教えてくれます。
**質問する勇気だけ**が必要です。
