# 02. ピタゴラスの定理 — 家を建てる人の、最強の道具

**結論を先に**: 直角三角形の3つの辺には、必ず `a² + b² = c²` という関係がある。

「だから何？」と思いますよね。
実はこれ、**GPS・スマホの画面・建築・ゲームのキャラ移動まで、ありとあらゆる場面**で使われている、人類で**最も実用的な式**のひとつです。

## まず、絵で理解する

直角三角形（一つの角が90度の三角形）を思い浮かべます。

```
       │\
       │ \
     b │  \  c  (斜辺、いちばん長い辺)
       │   \
       │    \
       └─────
          a
```

- 縦の辺 = `b`
- 横の辺 = `a`
- 斜めの辺 = `c`（**斜辺**、いつも一番長い）

ピタゴラスの定理が言うのは:

> **「縦²」と「横²」を足すと、必ず「斜辺²」になる**

数式で書くと:

```
a² + b² = c²
```

## 具体例で確認

横3、縦4の直角三角形があったら、斜辺は？

```
3² + 4² = c²
9 + 16  = c²
25      = c²
c       = 5    ← 平方根（√25 = 5）
```

つまり斜辺は **5** です。これを「**3-4-5 の三角形**」と言って、大工さんが直角を出すときに今でも使う、4000年前から知られている形です。

## 何の役に立つの？

### 🏠 (1) 大工さんが直角を出す道具

「ここを直角90度にしたい」というとき、メジャーで `3m と 4m` を測って、対角が `5m` ぴったりになれば、それは正確に直角。
**スマホの角度計が無かった時代から人類が使ってきた技**。

### 📱 (2) スマホ画面のサイズ

「6.1インチのiPhone」と言うとき、その6.1インチは**斜辺の長さ**。
本体の縦と横の長さから、ピタゴラスの定理で計算されています。

### 🗺️ (3) 地図上の2点間の距離

地図上で「家から駅まで、まっすぐ最短ルートで何メートル？」を計算するとき:

```
距離 = √( (横の差)² + (縦の差)² )
```

これ、ピタゴラスの定理を解いた形そのもの。
GPS も、これを3次元に拡張して使っています。

### 🎮 (4) ゲームのキャラクター移動

ゲームで「敵キャラまでの距離」を計算するときも、座標 (x1, y1) と (x2, y2) の距離は:

```
距離 = √((x2−x1)² + (y2−y1)²)
```

完全にピタゴラスの定理。
RPGの「敵がプレイヤーに近づくと攻撃する」処理は、これで動いています。

### 🤖 (5) 機械学習の「類似度」

なんと、**機械学習でも使われています**。
「2つのデータがどれだけ似ているか」を測るときに、データを多次元のベクトルとして見て、ピタゴラスの定理を一般化した距離（**ユークリッド距離**）を計算します。

```python
# 2つのデータ点の距離
import numpy as np

a = np.array([1, 2, 3])    # データ1
b = np.array([4, 6, 8])    # データ2

distance = np.sqrt(np.sum((a - b) ** 2))   # ピタゴラスの一般化
print(distance)            # 7.07...
```

つまり、**ピタゴラスの定理さえ理解できれば、AI の「類似度」の半分は分かったも同然**です。

## Python で遊んでみる

```python
import math

def pythagoras(a: float, b: float) -> float:
    """直角三角形の2辺から斜辺を求める."""
    return math.sqrt(a ** 2 + b ** 2)

print(pythagoras(3, 4))     # 5.0
print(pythagoras(5, 12))    # 13.0
print(pythagoras(1, 1))     # 1.414...  ← √2 = ルート2、無理数の代表
```

## 補足: 「ルート」 √ ってなに？

`√25 = 5` というのは、「**2乗すると 25 になる数**」を意味します。
`5 × 5 = 25` なので、`√25 = 5`。

`√2` のように整数にならないものは**無理数**と呼ばれ、永遠に小数が続きます（1.41421356...）。
紀元前のピタゴラス教団は、無理数の存在を知って大ショックを受けた、という逸話があります（詳しくは [`columns/01_pythagoras_story.md`](columns/01_pythagoras_story.md)）。
