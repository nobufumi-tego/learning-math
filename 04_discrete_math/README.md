## キーとなる Python ツール

```python
import math
from itertools import permutations, combinations

# n!
math.factorial(5)              # 120

# 組合せ ₙCₖ
math.comb(5, 2)                # 10

# 順列 ₙPₖ
math.perm(5, 2)                # 20

# 全順列を列挙
list(permutations([1, 2, 3]))  # [(1,2,3), (1,3,2), ...]

# 全組合せを列挙
list(combinations([1, 2, 3, 4], 2))  # [(1,2), (1,3), ...]
```

## 重要な記法

| 記号 | 読み方 | 意味 |
|---|---|---|
| `n!` | n の階乗 | 1 × 2 × ... × n |
| `ₙCₖ` または `C(n,k)` | n choose k | n個から k 個選ぶ組合せ数 |
| `(n k)` (縦) | binomial | 二項係数 |
| `ₙPₖ` | n permute k | n個から k 個並べる順列数 |

組合せ数の公式:
```
C(n, k) = n! / (k! (n−k)!)
```
